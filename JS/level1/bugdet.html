예산

문제 풀이 방법

  1. 예산이 적게 들어가는 부서를 많이 지원해주는 것이 최대 지원값을 뽑아내기에 유리하다.
  2. 부서별 신청 금액이 담긴 배열을 오름차순으로 정렬한다.
  3. 정렬된 배열 값을 예산과 비교해 예산보다 신청 금액이 작으면 지원한다.

코드

        
            function solution(d, budget) {
                var answer = 0;
                
                d.sort((a, b) => a - b);
                d.forEach(v => {
                    // console.log(v + ' ');
                    if(budget - v > -1) {
                        budget -= v;
                        answer++;
                    }
                })
                
                return answer;
            }
