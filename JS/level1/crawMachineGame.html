크레인 인형뽑기 게임

문제 풀이 방법

  1. 인형을 1회 뽑을 때 해당 열에 대해 행 인덱스가 증가하면서 배열의 값을 확인한다.
  2. 인형을 뽑은 경우 기존에 뽑은 인형과 종류가 같은지 비교하고 다음 로직에 따라 처리한다. 같은 인형이면 두 인형을 지운 뒤 +2점, 다른 인형이면 배열에 마지막 요소로 넣어준다. 보드에서 인형이 있었던 배열의 값을 0으로 바꾸어준다.
  3. 위와 같은 방식을 모든 움직임(move 배열)에 대해 실행한다.

코드

        
            function solution(board, moves) {
                var answer = 0;
                var array = [];
                var value = 0;
            
                moves.forEach(v => {
                    for(var i in board) {
                        value = board[i][v - 1];
                        if(value) {
                            if(array[array.length - 1] === value){
                                array.pop();
                                answer += 2;
                            } else {
                                array.push(value);
                            }
                            board[i][v - 1] = 0;
                            break;
                        }
                    }
                });
                
                return answer;
            }
