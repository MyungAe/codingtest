폰켓몬

문제 풀이 방법

  1. 배열 하나를 생성한 뒤, nums 배열의 값을 인덱스로 사용해 만든 배열의 값을 1로 고정시킵니다.
  2. 그 후 배열에서 값이 1인 인덱스의 개수를 추출해 nums 배열의 크기의 반보다 작은지 비교합니다.
  3. 만약 nums의 반보다 크다면 nums의 배열 크기의 반을 리턴합니다.

코드

문제 풀이 코드는 다음과 같습니다.

        
        function solution(nums) {
            var sortArray = new Array();
            var slotSize = 0;
            var index = 0;
            
            for(var k = 0; k < nums.length; k++){
                sortArray[k] = 0;
            }
            
            for(var i = 0; i < nums.length; i++){
                index = nums[i] - 1;
                sortArray[index] = 1;
            }
            
            for(var j = 0; j < sortArray.length; j++){
                if(sortArray[j] == 1) slotSize++;
            }
            
            var answer = slotSize;
            if(answer > nums.length / 2) answer = nums.length / 2;
            
            return answer;
        }
